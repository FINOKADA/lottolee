# 로또리 (LottoLEE)

로또 분석 및 예측 애플리케이션

## 주요 기능
- 로또 번호 예측
- 당첨 번호 분석
- 개인화된 통계

## 개발 환경
- Flutter 3.x
- Dart 3.x

## 프로젝트 구조
```lib/
   ├── screens/      # 화면 UI 파일들
   ├── widgets/      # 재사용 가능한 위젯들
   ├── models/       # 데이터 모델
   ├── services/     # API, 데이터베이스 등 서비스
   ├── utils/        # 유틸리티 함수들
   ├── providers/    # 상태 관리
   ├── constants/    # 상수값들
   └── main.dart     # 앱의 시작점
